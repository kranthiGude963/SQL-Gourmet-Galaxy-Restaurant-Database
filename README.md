# SQL-Gourmet-Galaxy-Restaurant-Database
Relational database design project for a restaurant management system including table creation, primary and foreign key relationships, dietary restrictions mapping, and sample data insertion using SQL Server.

Gourmet Galaxy Restaurant Database (SQL Project)
** Project Overview**

This project demonstrates relational database design and SQL implementation for a restaurant management system.

The database includes restaurants, menu items, and dietary restrictions, designed using SQL Server.
